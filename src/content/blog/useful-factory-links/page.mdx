## What this list covers

This post is a scannable shortlist of factory-focused destinations for
you-agent-factory readers. It is not a second docs index, not a Model Atlas or
attention-reference dump, and not a benchmark or paper-download mirror.

The groups below organize the shortlist: CLI and repository entry points,
harnesses, workflow techniques, MCP and dynamic workflows, and a short set of
ecosystem readings. Prefer the published docs routes in related reference pages
when you need the stable definition.

## CLI and repository

Start with the [CLI documentation](/docs/documentation/cli) for install and
first-run commands, and keep the repository nearby when you check in workflow
and agent files.

## Harnesses

Use [harness support](/docs/documentation/harness-support) when you need to know
which agent runtimes the factory can drive and how those attachments behave in a
long-running run.

## Workflow techniques

Reach for technique pages when the work is a loop, a worktree split, compaction,
or a writer-reviewer pass rather than a one-shot prompt.

## MCP and dynamic workflows

Use [MCP documentation](/docs/documentation/mcp) when tools arrive through the
Model Context Protocol, and
[dynamic workflows](/docs/documentation/dynamic-workflows) when the workflow
shape itself is assembled at runtime.

## Relevant ecosystem reading

Keep a short set of factory-relevant external reads for context outside this
docs site—CLI and repository hubs, agent-runtime ecosystems, and
workflow-pattern writing that stays useful for operators.

## Related reference pages

Start with the product overview at
[What is you-agent-factory](/docs/documentation/what-is-you-agent-factory).
CLI, harness support, MCP, and dynamic workflows docs are linked above; the
harness concept page is linked below.
