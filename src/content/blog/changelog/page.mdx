## Recent releases

### you-agent-factory v0.0.5 — 2026-06-02

**Version:** v0.0.5  
**Released:** 2026-06-02

you 0.0.5 is the latest published product release. For operators, the release
focuses on configuration, multi-session factory work, and day-to-day CLI
ingress:

- Configure the factory directly from the website instead of only through local
  file edits.
- Fix website system-state rendering bugs that made live factory status harder
  to trust.
- Standardize interface names and configuration chrome so the UI reads as one
  consistent surface.
- Add CLI support for enumerating work, moving work, submitting work, and
  querying sessions.
- Keep multiple factory sessions in one instance and deploy to multiple
  workspaces concurrently.

Install or upgrade with the paths on the
[install documentation](/docs/documentation/install) page, then use the
[CLI documentation](/docs/documentation/cli) teaching matrix or the
[CLI command index](/docs/documentation/cli-command-index) for the day-to-day
`you` commands those release notes describe. The raw tag and assets are on
[v0.0.5 on GitHub Releases](https://github.com/portpowered/you-agent-factory/releases/tag/v0.0.5).
