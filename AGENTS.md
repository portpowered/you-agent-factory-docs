This project is the you agent factory's docs pages, which is an agent orchestrator function. 

The raw codebase for the you agent factory can be found at https://github.com/portpowered/you-agent-factory

The project is intended to both document the factory's functionality, support installation, but also to support content for search and helpful tips and guides.

## architecture
The technology choices are in docs/internal/technology.md
the system goals are in the customer-system-goals.md

Changes to the system MUST comply to towards teh target technology choices and system goals. 
## implementation
Code implementation of the you agent factory docs is done autonomously via the you agent factory. 

## standars

Appropriate code standards and structuring standards can be found under factory/docs/standards.

Agents MUST follow the appropriate standards when performing related changes. 

see factory/docs/standards/code-review-standards for code review. 
see factory/docs/standards/general-website-standards.md for websites.
see factory/docs/standards/planning-standards.md for planning.
