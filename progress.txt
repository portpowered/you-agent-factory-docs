## Codebase Patterns
- Reusable docs primitives live under `src/components/docs/primitives/` and are re-exported from `index.ts`.
- Shared reviewer labels, fixture data, and example route constants belong in `src/lib/docs-primitives.ts`.
- `formatCodeBlockLanguageLabel()` maps language slugs to reviewer-visible labels for `CodeBlock` figcaptions.
- `CodeBlock` is a client component (`useId` + `aria-labelledby`) so figcaption names are exposed to assistive tech and happy-dom tests.
- Docs chrome is shared through `src/components/docs/docs-shell-layout.tsx`; pass `activeNav` and render page content as children.
- The reviewer-visible code presentation example is at `/docs/examples/code-presentation/` via `src/app/docs/examples/code-presentation/page.tsx`.
- Primitive styles extend `src/app/globals.css` using existing `--landing-*` CSS variables.
- Component tests for docs surfaces belong in `tests/unit/docs-shell.test.tsx`; focused `CodeBlock` tests belong in `tests/unit/code-block.test.tsx`; static export route checks belong in `tests/unit/static-export.test.ts`.
- Manual browser verification serves `out/` under `/you-agent-factory-docs` per `docs/internal/processes/manual-qa.md`.
